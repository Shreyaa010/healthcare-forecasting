# healthcare-forecasting
Healthcare analytics dashboard with prediction using Flask

## Project Description
This project is a web-based healthcare analytics dashboard built using Flask. It visualizes historical data and predicts future care load using machine learning.

## Features
- Data cleaning and preprocessing
- Interactive dashboard
- Data visualization
- Prediction using Linear Regression
- Data filtering and download option

## Technologies Used
- Python
- Flask
- Pandas
- Matplotlib
- Scikit-learn

## How to Run
1. Install required libraries
2. Run `python app.py`
3. Open http://127.0.0.1:5000
